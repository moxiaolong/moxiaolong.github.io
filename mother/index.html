# 母亲（mother）

拉康认为，"母亲"（The Mother）不仅仅是指生物学上的母亲，而是一个象征性的角色，代表了个体最初的情感和身份的来源。在婴儿阶段，母亲是婴儿的一切，她提供了食物、温暖、安全感和情感满足。这一阶段，婴儿无法区分自己和母亲，它们认为自己和母亲是一体的，这种状态被称为"镜像阶段"。母亲的爱和关怀塑造了婴儿的自我认同，但同时也为以后的心理冲突埋下了种子。

拉康强调了母亲的象征性意义，将其视为符号系统的一部分，不仅包括语言，还包括文化和社会规范。母亲的象征性功能在个体主体的发展中起到关键作用，她传递了文化和社会的价值观念，并与儿童建立了符号化的关系。母亲的象征性不仅体现在言语上，还体现在身体和行为上，对于个体主体性的形成至关重要。

# 母亲的匮乏

这一概念指的是在个体的心理发展过程中，婴儿最初与母亲建立的情感联系和依赖关系逐渐被打破，婴儿开始认识到母亲不能满足他所有的欲望和需求。这种母亲的匮乏意味着婴儿首次体验到自己的限制和依赖性。

# 发现母亲的匮乏，认为父亲拥有菲勒斯，认同父亲，接受象征性阉割

在精神分析中，孩子在早期与母亲建立密切的情感联系，认为母亲是一切的来源和满足。然而，随着孩子的成长，他开始认识到母亲并不是无穷无尽的满足源泉，母亲也有她自己的限制和需求。这一认识引发了孩子对自己和外界的更深思考，他开始寻求解决这一匮乏感的方式。

在这一过程中，孩子转向父亲，将他视为能够填补母亲匮乏的源泉。父亲代表了社会和文化中的权力和规范，他也是象征性的菲勒斯。通过认同父亲，孩子试图获得一种象征性的满足，即通过遵守社会和文化的规则来获得自我认同和满足感。

这个过程强调了象征性阉割的概念，孩子在这一过程中接受了自己无法完全满足欲望的事实，并通过认同父亲来获得一种社会和文化的认可。这种认同过程在精神分析中被认为对个体主体性的发展具有关键性影响，它涉及到性别认同、自我认同和社会化过程。这个过程也反映了弗洛伊德和拉康等精神分析家对个体心理发展的理论探索。

# 对拉康精神分析符号学的概述

图片中展示的是四个性化公式以及主体进入符号系统（学会说话）的各个阶段。我先对整个结构框架进行简述再细化各个阶段的特征。

1.母子共同体阶段：人类的婴儿都是早产儿，早产儿在出生后一段时期没有任何行动能力，一切需求都要依靠母亲（以及其他监护人）的帮助，在这种情况下婴儿会有全能感（母子共同体状态），这也就是拉康意义上的大他者A。

2.母子共同体分离：孩童在这个阶段在需求的满足上遭到失败（比如断奶后要自己吃饭），孩童会意识到母亲与自身的分离（全能的是母亲，自己是无能的，并且自身与母亲不是一体的而是分离的），这时孩童感到母子共同体状态的缺失/匮乏。

3.象征秩序的介入：面对与母亲分离的原处创伤，孩童会进入语言的维度，用“母亲”的语词填补实在母亲的缺失。这时就会产生原初能指S（/A），它没有所指，它指向的就是与母亲分离的匮乏状态，它的所指就是所指（实在母亲）的缺失。这时孩童就进入象征秩序了。

4.象征父亲的阉割：进入象征秩序的孩童想要回到母子共同体的状态，它（此时还没有性化）想成为母亲的欲望对象，但它发现母亲欲望着父亲，这就表明父亲身上有某个东西自己没有。这时婴儿就会去欲望母亲欲望的东西，也就接受了父性秩序的阉割。认识到自身匮乏的主体将自身标记为/S，他/她的欲望指向客体小a。

接下来详细描述各个阶段:

1.母子共同体状态是回溯性建构的幻想，婴儿从出生就与母亲是分离的，并不存在母子共同体状态（大他者A不存在，是符号系统回溯性建构的）。

2.语言给出的是匮乏，原初能指指向的就是所指的不在场。主体在接受阉割进入符号系统之后就从实在界进入象征界，但象征界并不能让主体回到母子共同体状态，父亲也并不能满足母亲的欲望，那么母亲的欲望指向什么？主体会将母亲的欲望对象设定为客体小a，a是符号化的剩余，是主体进入象征界时遗留在实在界的原质。

3.在意识到自身和语言的匮乏，将欲望对象锚定为客体小a后，主体会完成镜像认同并完成性化。

镜像认同：主体会将镜子中自身的图象认同为自身（以此完成对自己身体的整合以及对理想自我的建构），并通过母亲的视线和话语（看，那就是你）确证自身（同时也会将镜像自我误认为是客体小a去追求）。

性化秩序：面对符号系统的“乱伦禁忌”和“露yin禁令”，男性会认为自身的阳具是母亲的欲望对象，认同阳具能指的同时男性便会因害怕失去阳具而陷入阉割焦虑，同时将欲望投射在女性身上（认为女性有某种神秘本质，以女性为对立面构建自己的性别认同，组成男性共同体），对女性的追求就是对母亲的替代（乱伦禁忌）。男性的两个性化公式就是“所有人都服从符号秩序”和“至少存在一个人，不服从符号秩序”，符号秩序的例外就是女性的神秘本质（其实还有一个例外就是可以无限享乐的原父，但男性不会承认原父可以无限享乐，他们会认为原父是不享乐的，而将可以无限享乐的原父星象投射为女性的神秘本质）。这样男性就完成了阉割，他们的欲望就固着在女体和共同体认同上（对共同体的认同会与对理想自我的认同短路）。

而女性进入性化秩序接受的是加倍的阉割。因为女性在生物学上没有阳具，面对符号系统的“露yin禁令”，女性被迫要求“不准露出你没有的东西”，但女性会认为并不会将阳具的缺失把握为一种缺失，她们会认为这只是生物学上的差异，生物学上的阳具并不是大他者的化身。所以女性虽然被加倍的阉割，但她们能感受到符号秩序的荒谬，实际上被阉割的程度更轻。女性的两个性化公式是对男性秩序的直接否定：“并非所有人都服从符号秩序”和“并非至少存在一个人，不服从符号秩序”。女性不承认男性秩序存在，不承认男性以女性为对立面建立身份认同的方式本身，但仍会认为有一套秩序存在。

4.欲望就是围绕着客体小a建构的，欲望=需要-需求，其内核是对不可能性的追求（客体小a本身是回溯性建构）。
