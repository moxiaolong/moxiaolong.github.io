Pip包的离线下载和安装

用pycharm打开项目，新建一个空的requirements.txt，在源代码文件import处会提示加入requirements.txt。

加入后执行 pip download package -r .\requirements.txt

离线下载完成。

pip install -r requirements.txt --no-index --find-links=package 安装离线下载的包
